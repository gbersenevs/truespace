## 4️⃣ npm install - Установка зависимостей

### Команда:
```bash
npm install
```

### Что увидите (будет прокручиваться):

```
npm WARN deprecated inflight@1.0.6: ...
npm WARN deprecated glob@7.2.3: ...

added 512 packages, and audited 513 packages in 1m

125 packages are looking for funding
  run `npm fund` for details

found 0 vulnerabilities
```

**Это займет 2-3 минуты. Подождите!**

✅ **УСПЕХ:** Видите `added XXX packages`

❌ **ОШИБКА:** 
```
npm ERR! code ENOENT
npm ERR! syscall open
npm ERR! path /Users/.../package.json
```
→ **Решение:** Вы не в папке проекта. Вернитесь к Шагу 1.

```

```
→ **Решение:** Порт 3000 занят
```bash
lsof -ti:3000 | xargs kill -9
npm run dev
```

```
MongoServerError: bad auth
```
→ **Решение:** Это нормально! БД настроим на следующем шаге.

## 📸 Полезные горячие клавиши

### В терминале:
- `Ctrl + C` - остановить сервер
- `Cmd + K` - очистить терминал
- `Cmd + T` - новая вкладка терминала

### В браузере:
- `Cmd + R` - обновить страницу
- `Cmd + Shift + R` - обновить без кэша
- `Cmd + Option + I` - открыть DevTools

### В VS Code:
- `Cmd + P` - быстрый поиск файлов
- `Cmd + B` - показать/скрыть sidebar
- `Cmd + J` - показать/скрыть терминал
